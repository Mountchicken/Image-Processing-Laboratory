# Image-Processing-Laboratory
-一款基于matlab app 的图像处理软件。拥有多通道图显示功能，可显示傅里叶变换图，直方图，离散余弦变换图。
-操作包括有对比度调整，各种滤波，直方图均衡，以及各种频域处理
